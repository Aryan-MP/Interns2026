# Internship Training Tasks Repository

## Overview
This repository is maintained by the training team to share **daily/weekly tasks, assignments, and practice exercises** with interns during the internship period.

All interns have been added as **contributors** to this repository to ensure smooth collaboration, version control practice, and real-world GitHub workflow exposure.

---

## Purpose of This Repository
- Share internship tasks and learning activities
- Help interns practice in a professional setup
- Encourage  documentation, and discipline

---

## Instructions for Interns
Please follow the guidelines below strictly:

- Do **not** push directly to the `main` branch
- Create a **separate branch** using your name
